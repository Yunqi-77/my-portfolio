# 🌐 Personal Portfolio Website

A responsive personal portfolio website built with **HTML**, **CSS**, and **JavaScript** to showcase my professional background, technical skills, and software development projects.

The website serves as a central hub for my work, highlighting featured projects, providing information about my experience, and allowing visitors to learn more about me.

# 🚀 Running Locally

Clone the repository:

```bash
git clone https://github.com/Yunqi-77/my-portfolio.git
```

Open the project folder:

```bash
cd my-portfolio
```

Open `index.html` in your preferred web browser.

Alternatively, use the Visual Studio Code **Live Server** extension for local development.
